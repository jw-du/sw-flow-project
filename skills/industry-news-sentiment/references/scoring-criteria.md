# 评分标准

## 详细评分指南

### 0 分——负面或中性

#### 监管打击
- **严查**：对公司或行业发起调查
- **罚款与处罚**：金钱处罚、制裁
- **行政措施**：业务暂停、执照吊销
- **合规要求**：新规提高合规负担

**指标关键词**：
- “严查”
- “罚款”
- “处罚”
- “整顿”
- “约谈”
- “监管”
- “违规”
- “退市”
（暂停）

#### 无相关新闻
- 常规行政更新
- 轻微运营调整
- 不影响市场的信息

### 0.5 分——中等影响

#### 行业常规新闻
- 行业报告与统计
- 市场趋势分析
- 板块表现更新

#### 公司常规新闻
- 季度财报（非异常）
- 管理层变动（非核心/战略）
- 产品发布（非突破）
- 合作关系公告

#### 市场一般新闻
- 分析师观点与预测
- 市场评论
- 技术分析
- 投资者情绪调查

**指标关键词**：
- “公告”
- “报告”
- “会议”
- “计划”
- “预期”
- “调整”
- “优化”
- “改进”

### 1 分——重大正面影响

#### 重大货币政策
- **降准**：银行可增加放贷
- **降息**：降低融资成本
- **流动性投放**：央行向市场提供流动性

#### 政府支持
- **国家队增持**：国家资金入市
- **平准基金**：稳定市场的政府基金
- **政策支持**：对行业/市场的明确扶持

#### 重大财政刺激
- 大规模基建投入
- 显著减税
- 财政刺激方案
- 发行政府债以刺激

#### 系统性支持
- 全市场支持举措
- 系统性风险防范
- 金融稳定措施

**指标关键词**：
- “降准”
- “降息”
- “增持”
- “平准基金”
- “刺激”
- “减税”
- “流动性”
- “支持”
- “稳定”

## 评分决策树

```
开始
  │
  ├─ 是否有监管打击？
  │   └─ 是 → 0 分
  │
  ├─ 是否有重大政策支持？
  │   └─ 是 → 1 分
  │
  ├─ 是否为行业/公司常规新闻？
  │   └─ 是 → 0.5 分
  │
  └─ 是否无相关新闻？
      └─ 是 → 0 分
```

## 加权因子

### 新闻来源可信度
- **官方政府来源**：满权重（1.0x）
- **主流财经媒体**：高权重（0.9x）
- **公司公告**：高权重（0.9x）
- **分析师报告**：中等权重（0.7x）
- **社交媒体/传闻**：低权重（0.3x）

### 时效与紧迫性
- **突发新闻**：满分
- **预定公告**：满分
- **延迟报道**：0.9 倍分数
- **历史背景**：按相关性调整

### 市场环境
- **牛市**：正面新闻影响放大（+10%）
- **熊市**：正面新闻更受重视（+15%）
- **高波动**：所有新闻权重提高（+20%）

## 行业调整因子

### 房地产
- 政策公告：更高权重（1.2x）
- 监管变动：更高权重（1.2x）
- 市场数据：标准权重

### 科技
- 创新突破：更高权重（1.3x）
- 监管动作：更高权重（1.2x）
- 公司新闻：标准权重

### 金融
- 货币政策：最高权重（1.5x）
- 监管变动：更高权重（1.3x）
- 市场数据：标准权重

### 医药
- 药品获批：更高权重（1.3x）
- 政策变动：更高权重（1.2x）
- 临床试验：中等权重（0.8x）

### 新能源
- 政策支持：更高权重（1.3x）
- 技术突破：更高权重（1.2x）
- 市场数据：标准权重
