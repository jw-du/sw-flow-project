# 行业新闻情绪

## 概述

分析行业新闻对市场情绪的影响，并给出用于投资决策的量化分数。本技能提供标准化评分标准、关键词识别与行业语境，帮助评估新闻相关性与市场影响。

## 评分标准

### 0 分——负面或中性
- **无相关新闻**：无影响市场的信息
- **监管打击**：严查、罚款、处罚
- **偏空监管动作**：收紧监管、提高合规要求

**关键词**：严查, 罚款, 处罚, 整顿, 约谈, 监管, 违规, 退市, 暂停

### 0.5 分——中等影响
- **行业常规新闻**：例行更新、轻微公告
- **公司常规新闻**：财报、管理层变动
- **市场一般新闻**：市场点评、分析师观点

**关键词**：公告, 报告, 会议, 计划, 预期, 调整, 优化, 改进

### 1 分——重大正面影响
- **重大货币政策**：降准，降息
- **政府支持**：国家队增持，平准基金
- **重大财政刺激**：大规模刺激方案、减税
- **系统性支持**：流动性投放、政策支持

**关键词**：降准, 降息, 增持, 平准基金, 刺激, 减税, 流动性, 支持, 稳定

### 行业特定事件

对于行业级重大政策公告（如房地产、科技、医药）：

**格式**：分数 + 行业标签

**示例**：
- “1.0 + 房地产” - 房地产重大政策
- “0.5 + 科技” - 科技行业中等影响新闻
- “0.0 + 金融” - 金融板块负面新闻

**重点行业**：房地产, 科技, 金融, 医药, 新能源, 消费, 制造, 基建

## 分析流程

### 步骤 1：识别新闻类型

确定新闻类别：

1. **监管/政策**：政府动作、法规、政策变化
2. **公司**：公司公告、财报、管理层信息
3. **市场**：全市场评论、分析师观点、趋势
4. **行业特定**：板块级新闻、行业趋势

### 步骤 2：检查关键词

在新闻文本中查找上述评分标准中的关键词：

- 若出现 0 分关键词 → 记 0 分
- 若出现 1 分关键词 → 记 1 分
- 若出现 0.5 分关键词 → 记 0.5 分

### 步骤 3：评估影响级别

评估影响强度与时效性：

- **即时市场影响**：数小时/数日内影响价格
- **长期结构性影响**：改变基本面的政策
- **行业级 vs 个股级**：影响范围越广权重越高

### 步骤 4：判定行业背景

如为行业特定新闻：

1. 确定主要受影响的行业
2. 判断是否为重大政策或例行新闻
3. 在分数后标注行业标签

### 步骤 5：输出格式

给出结构化输出：

```
评分: [0 | 0.5 | 1]
类别: [监管 | 公司 | 市场 | 行业特定]
行业: [行业名称或 N/A]
关键要素: [识别出的关键信息]
影响评估: [对市场影响的简述]
```

## 特殊情形

### 多条新闻

当分析多条新闻时：

1. 分别为每条打分
2. 聚合分数：
   - 分数求和（例如 3 条范围 0–3）
   - 按时效加权（越近权重越高）
   - 考虑累计影响

### 相互冲突的信号

当新闻同时包含正负因素：

1. 识别主导信号
2. 记录冲突因素
3. 基于净影响给出分数
4. 给出决策理由

### 突发 vs 传闻

- **官方公告**：按满分处理
- **传闻/猜测**：按 0.5 倍计分（折扣）
- **传闻被证实**：确认后按满分处理
