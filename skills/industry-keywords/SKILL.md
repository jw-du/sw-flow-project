# 行业关键词库

## 使用方式

```bash

kimi skill run bing_search --query "房地产 放松限购 政策支持"
kimi skill run bing_search --query "科技 芯片国产化 技术突破"
```

## 搜索优化策略

### 推荐：少关键词 + 多结果 + 多轮搜索

经过实践验证，以下策略可获得更全面的行业信息：

**❌ 避免：关键词过多**
```bash
# 不推荐：关键词堆砌，容易遗漏结果
kimi skill run bing_search "房地产 放松限购 政策支持 2026年3月 最新" 5
```

**✅ 推荐：精简关键词 + 增加结果数 + 多组搜索**
```bash
# 推荐：每组2-3个关键词，请求10条结果，多组覆盖

# 房地产 - 第1组
kimi skill run bing_search "房地产 放松限购" 10
# 房地产 - 第2组  
kimi skill run bing_search "房地产 保交楼" 10
# 房地产 - 第3组
kimi skill run bing_search "房贷利率 下调" 10
```

### 为什么这样优化？

| 策略 | 效果 |
|-----|------|
| **少关键词**（2-3个） | 扩大匹配范围，避免过度筛选 |
| **多结果**（10条） | 获取更多信源，交叉验证 |
| **多组搜索**（3-5组/行业） | 覆盖不同角度，减少遗漏 |

## 与 industry-news-sentiment 搭配使用

获取新闻后，使用 `industry-news-sentiment` skill 进行情绪评分：

| 新闻类型 | 情绪得分 | 说明 |
|---------|---------|------|
| 降准/降息/国家队增持 | 1.0 | 重磅利好 |
| 行业扶持政策 | 0.8 | 重要利好 |
| 普通政策/公告 | 0.5 | 中性 |
| 监管收紧/处罚 | 0.0 | 利空 |

完整流程示例：
```bash
# Step 1: 使用关键词搜索
kimi skill run bing_search "房地产 保交楼 政策 2026年3月"

# Step 2: 情绪分析（基于搜索结果）
# 根据 industry-news-sentiment 评分标准判断
# - 有重磅政策（降准/专项债）→ 1.0分
# - 普通支持政策 → 0.5分
# - 无消息 → 0.3分
```
